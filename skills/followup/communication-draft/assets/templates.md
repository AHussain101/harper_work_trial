# Communication Style Guide

**Note: These are NOT copy-paste templates.** Use them as starting points, then personalize based on the client's communication style from their emails, calls, and texts.

## Style Spectrum

Match your communication to where the client falls on these spectrums:

```
Formal ←――――――――――――――――→ Casual
"Dear Mr. Smith"          "Hey John!"

Detailed ←――――――――――――――→ Brief  
Full explanations          Bullet points only

Professional ←――――――――――→ Friendly
Business-focused           Personal touches

Cautious ←―――――――――――――→ Direct
Multiple options           Clear recommendation
```

## Analyzing Client Style

### Email Indicators

| They Write | They Prefer |
|------------|-------------|
| "Dear [Name]" | Formal greeting |
| "Hi!" or "Hey" | Casual greeting |
| Long paragraphs | Detailed explanations |
| Bullet points | Concise information |
| Quick responses | Fast turnaround |
| Days to respond | Not urgent, no pressure |
| Questions about details | Thorough coverage info |
| Questions about price | Bottom-line focus |

### Call Indicators

| They Do | They Prefer |
|---------|-------------|
| Get straight to business | No small talk |
| Chat about personal life | Relationship building |
| Ask many questions | Detailed explanations |
| Say "sounds good" quickly | Quick, confident decisions |
| Say "let me think about it" | Time and space |
| Mention budget concerns | Price-focused options |
| Mention past claims | Risk-focused coverage |

### Text Indicators

| Pattern | Meaning |
|---------|---------|
| One-word replies | Keep texts brief |
| Full sentences | OK with longer texts |
| Emojis | Casual tone OK |
| Business hours only | Respect work/life |
| Quick responses | Preferred channel |

## Structure by Client Type

### The Busy Executive
- Subject line: Key info upfront
- Body: 3 sentences max
- Close: Single clear action

### The Detail Seeker
- Subject line: Topic + context
- Body: Full explanation with specifics
- Close: Offer to discuss further

### The Relationship Builder
- Subject line: Personal + professional
- Body: Personal check-in, then business
- Close: Warm sign-off

### The Price Shopper
- Subject line: Include numbers
- Body: Cost comparison, value explanation
- Close: Competitive positioning

## Channel Selection

Choose channel based on client behavior:

| Client Pattern | Use This Channel |
|----------------|------------------|
| Responds to emails same day | Email |
| Emails go unanswered, takes calls | Call |
| Texts back immediately | SMS |
| Prefers scheduled calls | Call with calendar link |
| Multiple decision makers | Email (can be forwarded) |

## Personalization Checklist

Before sending, verify:

- [ ] Greeting matches their style
- [ ] Length matches their typical messages
- [ ] Tone matches their formality level
- [ ] Referenced something specific from our conversations
- [ ] Addressed their stated concerns
- [ ] Channel matches their preference
- [ ] CTA is clear and appropriate for their decision style
